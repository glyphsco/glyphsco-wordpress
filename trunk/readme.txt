== Description ==

A basic plugin with a Kit ID option. This adds the Glyphs.co javascript to the frontend and backend header, essentially installing Glyphs.co onto your Wordpress site. It also allows you to use the tags for icons you've added to your kit with a [glyphs] shortcode. 

**Requirements**

A Glyphs.co account - free or premium. Visit https://glyphs.co to get started.

**Coming Soon**

* View all available icons in your kit
* Find, add & remove icons in your kit directly through the plugin
* Find & place Glyphs.co photos directly through the plugin


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/glyphs-company` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Glyphs.co screen to configure the plugin with your 6-digit Glyphs.co Kit Id for your site (Link is in the main sidebar navigation of your WordPress backend.)
4. You can now use the [glyphs] shortcode in your posts and pages to use Glyphs.co Icons directly on your site


== Frequently Asked Questions ==

= Do I need a Glyphs.co Premium account? =

No, you do not need a Glyphs.co Premium account to use this Wordpress plugin, but you will be limited to using the icons that you have access to through Glyphs.co

= Which icons are available to use? =

You may use any icons that are added to your current kit that is associated with this site. To add or remove icons from your kit, go to https://glyphs.co/icons/ and login to view your kits and configure them.
